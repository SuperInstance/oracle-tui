# Oracle TUI

Agentic terminal dashboard for the SuperInstance fleet.

Shows fleet health, crate status, nebula reflexes, and system info in a real-time terminal UI.

## Usage

```bash
node oracle-tui.js
```

## Commands
- `health` - Query fleet health
- `crates` - Show crate test status
- `reflexes` - List recent reflexes  
- `evolve` - Run evolution cycle
- `disk` - Show disk usage
- `quit` / `exit` - Exit
